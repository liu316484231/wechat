∞



不誇張地說，在冬天大白菜是東北人的本命。


隨著裝滿白菜的卡車駛進小區，大喇叭循環高喊：“白菜賤賣！！白菜賤賣！！白菜賤賣！！”大媽們肘挎編織袋、手推拉桿車、帶上自家大爺......東北的初冬以大爺大媽們擼袖子搶大白菜正式開始。


賣白菜的大哥也是頗有特色，講究的就是倆字：“豪邁！”什麼叫豪賣？就是五十斤以下不賣。記得我之前就在大卡車前要稱一顆白菜，喊了半天，大哥就是目光所及之處沒有你。現在想來那應該是個性溫和的大哥，實在不想跟年輕人理論一顆白菜夠乾啥的。


其實一方水土養一方人，上百年傳下來的東西是根深蒂固的。雖然我們這代年輕人已經沒有儲秋菜的習慣，但是看著家裡長輩囤白菜、曬白菜、腌酸菜，也是一件很有儀式感的事情。不可否認，這又是一個被大白菜支配的冬天。



白菜◦300g / 五花肉◦100g

泡發木耳◦100g / 泡軟粉條◦150g

蔥◦16g / 蒜◦2瓣 / 小米辣◦1根

花椒◦10顆 / 八角◦1個 / 薑◦4g

∆ 白菜幫斜切，即刀與白菜幫呈30度角切下去。斜切可以把白菜幫切薄，易熟也更入味兒。白菜葉手撕成塊，幫和葉分開裝。

∆ 提前將木耳泡發，粉條泡軟。五花肉、蒜切片，蔥切蔥花，薑切絲。



❶

鍋中無油，加入五花肉煸出油，煎至雙面焦黃。

∆ 帶皮五花做燉菜口感很Q彈，但豬皮在煸油的過程中會濺油。鍋蓋可以蓋上一半，切記不要全部蓋上，不然形成水蒸氣流進油鍋里....那就更熱鬧了。


❷

加入花椒、八角、蔥花、姜絲、蒜末，翻炒熗香。


❸

加入白菜幫翻炒，淋入一湯勺水，加蓋燜煮10分鐘，開蓋加入白菜葉翻炒均勻。


❹

加入十三香半茶匙，醬油1湯匙，蚝油2湯匙，鹽1茶匙，翻炒均勻，加入700ml清水。


❺

水沸後加入粉條和木耳，將粉條煮至無硬芯。

∆ 泡軟的粉條不耐煮，很快就熟了，要註意火候哦。


❻

嘗下湯汁，根據鹹淡適量加些鹽。最後加入小米辣，雞精半茶匙，出鍋~



爽滑勁道的紅薯粉條吸飽了鮮醇湯汁，嗦上一口……嘶，果然是迎接冬天的必備味道！！！
